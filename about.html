About This Blog

.NET, C#, automation, and real-world developer workflows.

Welcome to .NET Blog with Elvin, a developer-focused space where I explore modern .NET, C#, and backend engineering.

This blog is powered by an automated .NET pipeline that continuously discovers trending topics from the .NET ecosystem and generates posts using AI, helping to keep content current and relevant.

How the blog works

  • Collects .NET–related topics from developer communities.
  • Clusters them into 10 relevant themes.
  • Chooses one topic at random.
  • Generates a post using Gemini AI.
  • Creates a pull request with the new article.

Tech stack

  • .NET 8 — automation engine & GitHub integration
  • GitHub Actions — scheduling and CI/CD
  • Gemini AI — AI topic analysis and article generation
  • Jekyll + Minima — static site generation
  • GitHub Pages — hosting
